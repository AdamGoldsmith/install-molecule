## Demo-magic script to install molecule

### Requirements

Git

`sudo yum install git`

[demo-magic](https://github.com/paxtonhare/demo-magic.git)

`git clone https://github.com/paxtonhare/demo-magic.git`

### Running it

```
git clone
cd install-molecule
./install-molecule.sh
```


### Tested on

* Centos 7.4
